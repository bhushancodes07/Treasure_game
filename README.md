# 🎮 Treasure Game (C Project)

This is a fun terminal-based C game that includes:

## 🧩 Games
- Hangman
- SpellBee Word Game

## 🎯 Objective
Win both games to unlock the treasure chest!

## 🛠 Features
- Random word generation
- ASCII Hangman visuals
- Custom dictionary-based SpellBee game
- Story-based gameplay

## ▶️ How to Run

Compile:
gcc main.c -o game

Run:
./game
